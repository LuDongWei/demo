###mbbSku

##介绍
mbb商品sku模板生成器 

##页面分类
- Enter：入口  
- Sort：排序页面并判断商品状态  
- Data：数据页面，能对后端提供的数据进行修改并进行保存  
- template：模板页面，输入符合规则的模板生成代码

别的功能基本无效果，后来多次对sort页面进行了优化。
